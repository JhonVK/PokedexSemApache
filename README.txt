# Pókedex

Este projeto é uma Pókedex, uma aplicação que permite visualizar informações sobre diferentes Pokémons e armazená-los em diferentes estruturas de dados.

# Funcionalidades

- Listar todos os Pokémons
- Visualizar detalhes de um Pokémon específico
- Pesquisar Pokémons por nome

# Tecnologias Utilizadas

- Linguagem de Programação: JAVA
- Apache: Maeven
- API: PókeAPI
